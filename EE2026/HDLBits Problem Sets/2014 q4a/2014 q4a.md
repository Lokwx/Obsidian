Consider the _n_-bit shift register circuit shown below:

![Pasted image 20260710195403](Pasted%20image%2020260710195403.png)

```
module top_module (
    input clk,
    input w, R, E, L,
    output Q
);
	
    wire mux1, mux2;
    
    //Mux1
    always_comb begin
        case (E)
            1'b0: mux1 = Q;
            1'b1: mux1 = w;
        endcase
    end
    
    //Mux 2
    always_comb begin
        case (L)
			1'b0: mux2 = mux1;
            1'b1: mux2 = R;
        endcase
    end
    
    //DFF
    always @(posedge clk) begin
        Q <= mux2;
    end
    
endmodule
```

### Pointers 
1. No need to connect the output of the DFF back to Q because at the start Q is a fixed value, then once the DFF is triggered, the value of Q will be tied to the output of the mux2, which is D.
