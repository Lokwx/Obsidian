A JK flip-flop has the below truth table. Implement a JK flip-flop with only a D-type flip-flop and gates. Note: Qold is the output of the D flip-flop before the positive clock edge.

| J   | K   | Q         |
| --- | --- | --------- |
| 0   | 0   | Qold      |
| 0   | 1   | 0 (reset) |
| 1   | 0   | 1 (set)   |
| 1   | 1   | ~Qold     |

### JK Flip Flop (edge triggered)
1. Triggers by the clk (edge triggered)
2. J and K are the sync inputs that determine what the output is

![Pasted image 20260710201500](Pasted%20image%2020260710201500.png)

```
module top_module (
    input clk,
    input j,
    input k,
    output Q); 

    always @(posedge clk) begin
        if (j == 0 && k == 0) Q <= Q;
        else if (j == 0 && k == 1'b1) Q <= 0;
        else if (j == 1'b1 && k == 0) Q <= 1'b1;
        else Q <= !Q;
    end
    
endmodule
```
